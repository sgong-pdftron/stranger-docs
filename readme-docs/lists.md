## Rules

* Use ending period for list items that are complete sentences.
* Use numbered lists for sequences.
* Use bulleted lists for most other lists (use `*` for syntax).

### Example 1

```
1. Copy files.
2. Run the script.
```

1. Copy files.
2. Run the script.

### Example 2

```
* Apple
* Banana
```

* Apple
* Banana
